Tip Calculator

Screenshot

Tech Stack React, Vite, Tailwind CSS, JavaScript

This project is a simple tip calculator that allows users to input their bill amount, select a tip percentage, and see the calculated tip and total amount. It features a clean and responsive design, making it easy to use on both desktop and mobile devices. Looking back, if there was something I would change or do differently for this project, I would implement a history feature that saves users calculations and allows them to view past entries. This would allow users to revisit and keep track of their tipping habits.

Collector's Notebook

Screenshot

Tech Stack React, Vite, Tailwind CSS, JavaScript

This project is a collection tracker that allows users to create and manage lists of their collections (e.g., books, figurines, games). Users can add items to their collections, mark them as completed or in progress, and filter their lists based on status. The app is designed to be intuitive and visually appealing, with a focus on user experience. Looking back, if there was something I would change or do differently for this project, I would enhance the usability of the app by adding more organizational features, such as adding search capabilities within collections to find specific items quickly within large collections, and tags for categorizing and sorting related collections together.
